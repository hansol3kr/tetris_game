Blockfall — Windows 독립 실행판 (64-bit)
==========================================

■ 실행 방법
  1) 이 폴더를 통째로 원하는 위치에 복사합니다.
  2) Blockfall.exe 를 더블클릭합니다.

  * 처음 실행 시 "Windows의 PC 보호" (SmartScreen) 경고가 뜰 수 있습니다.
    서명되지 않은 앱이라서 그렇습니다. "추가 정보" → "실행"을 누르면 됩니다.

■ 폴더 구성 (반드시 함께 두어야 함)
  Blockfall.exe                         게임 본체 (리소스 내장)
  data_Blockfall_windows_x86_64\        .NET 어셈블리 (없으면 실행 안 됨)

  → Blockfall.exe 와 data_..._x86_64 폴더는 항상 같은 위치에 두세요.

■ 필요 환경
  - 64비트 Windows 10 / 11
  - .NET / Godot 설치 불필요 (전부 포함)

■ 조작 (키보드)
  ← / →  이동   ↓ 소프트드롭   Space 하드드롭
  ↑ / Z  회전   A 180도   C 홀드   Esc 일시정지
  (게임패드 지원)
